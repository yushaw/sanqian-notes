# 三千笔记 功能路线图

> 按优先级排序，编辑器功能优先

---

## 第一阶段：编辑器功能完善

### 1.1 高亮格式
**优先级**: P0 | **复杂度**: ⭐

**现状**: 不支持

**功能详情**:
- Markdown 语法: `==高亮文字==`
- 快捷键: `Cmd+Shift+H`
- 渲染为黄色背景
- 工具栏按钮

**实现**: Tiptap 自定义 Mark `Highlight`

---

### 1.2 Callout 提示块
**优先级**: P0 | **复杂度**: ⭐⭐

**现状**: 仅有普通引用块

**功能详情**:
- 语法: `> [!note]` `> [!tip]` `> [!warning]` `> [!danger]` `> [!info]`
- 支持的类型:
  | 类型 | 图标 | 颜色 |
  |------|------|------|
  | note | 📝 | 蓝色 |
  | tip | 💡 | 绿色 |
  | warning | ⚠️ | 黄色 |
  | danger | 🚨 | 红色 |
  | info | ℹ️ | 蓝色 |
  | quote | 💬 | 灰色 |
- 可折叠: `> [!note]-` 默认折叠, `> [!note]+` 默认展开
- 自定义标题: `> [!note] 自定义标题`

**实现**: Tiptap 自定义 Node，继承 blockquote

---

### 1.3 斜杠命令菜单
**优先级**: P0 | **复杂度**: ⭐⭐

**现状**: 仅支持 Markdown 快捷输入

**功能详情**:
- 输入 `/` 触发命令菜单
- 支持的命令:
  - `/h1` `/h2` `/h3` `/h4` - 标题
  - `/bullet` `/list` `/todo` - 列表
  - `/quote` `/callout` - 引用/提示块
  - `/code` - 代码块
  - `/divider` - 分割线
  - `/table` - 插入表格
  - `/image` - 插入图片
- 模糊搜索命令名称
- 键盘上下选择，回车确认

**实现**: Tiptap Suggestion 插件

---

### 1.4 大纲视图 / 目录 (TOC)
**优先级**: P0 | **复杂度**: ⭐⭐

**现状**: 无

**功能详情**:
- 侧边栏显示当前笔记的标题层级
- 实时更新
- 点击跳转到对应位置
- 显示层级缩进 (H1 > H2 > H3 > H4)
- 高亮当前可视区域的标题
- 快捷键切换 `Cmd+Shift+O`

**实现**: 监听编辑器内容变化，提取 heading 节点

---

### 1.5 拖拽排序块
**优先级**: P1 | **复杂度**: ⭐⭐⭐

**现状**: 不支持

**功能详情**:
- 鼠标悬停块左侧显示拖拽手柄 `⋮⋮`
- 拖拽块到其他位置
- 拖拽时显示插入位置指示线
- 支持拖入/拖出列表

**实现**: Tiptap + DragHandle 扩展

---

### 1.6 Toggle 折叠块
**优先级**: P1 | **复杂度**: ⭐⭐

**现状**: 不支持

**功能详情**:
- 可折叠/展开的内容块
- 点击箭头或标题切换状态
- 支持嵌套
- 斜杠命令 `/toggle` 插入

**实现**: Tiptap 自定义 Node

---

### 1.7 数学公式 (LaTeX)
**优先级**: P2 | **复杂度**: ⭐⭐

**现状**: 不支持

**功能详情**:
- 行内公式: `$E=mc^2$`
- 块级公式:
  ```
  $$
  \sum_{i=1}^n x_i
  $$
  ```
- 实时渲染预览
- 点击进入编辑模式

**依赖**: `katex`

---

### 1.8 Mermaid 图表
**优先级**: P2 | **复杂度**: ⭐⭐

**现状**: 不支持

**功能详情**:
- 代码块语言设为 `mermaid` 时渲染图表
- 支持: 流程图、时序图、甘特图、类图、饼图、思维导图
- 编辑时显示代码，失焦后显示图表

**依赖**: `mermaid`

---

### 1.9 附件支持
**优先级**: P2 | **复杂度**: ⭐⭐⭐

**现状**: 仅支持图片

**功能详情**:
- 支持拖拽/粘贴添加:
  - 视频: mp4, webm (内嵌播放器)
  - 音频: mp3, wav (内嵌播放器)
  - PDF: 缩略图预览
  - 其他文件: 图标 + 文件名
- 附件存储在应用数据目录
- 笔记删除时清理附件

**实现**: Tiptap 自定义 Node (video, audio, file)

---

### 1.10 Block 嵌入 (Transclusion)
**优先级**: P3 | **复杂度**: ⭐⭐⭐⭐

**现状**: 不支持

**功能详情**:
- 语法: `![[笔记名]]` 或 `![[笔记名#标题]]`
- 渲染被嵌入的内容（只读）
- 源内容更新时同步
- 有视觉边框区分

## 第二阶段：导航和搜索

### 2.1 反向链接面板
**优先级**: P0 | **复杂度**: ⭐⭐

**现状**: 数据已有，缺 UI

**功能详情**:
- 编辑器底部或侧边显示
- 列出所有链接到当前笔记的笔记
- 显示链接上下文（前后文字）
- 点击跳转
- 显示链接数量

---

### 2.2 快速切换 (Quick Switcher)
**优先级**: P0 | **复杂度**: ⭐⭐

**现状**: 无

**功能详情**:
- 快捷键 `Cmd+O` 或 `Cmd+K`
- 模糊搜索笔记标题
- 显示笔记本归属
- 上下键选择，回车打开
- 无输入时显示最近访问
- 可直接创建新笔记

---

### 2.3 搜索结果高亮
**优先级**: P0 | **复杂度**: ⭐

**现状**: 搜索后不高亮

**功能详情**:
- 搜索后打开笔记，滚动到第一个匹配
- 高亮所有匹配关键词
- 上一个/下一个导航
- Esc 清除高亮

---

### 2.4 高级搜索语法
**优先级**: P1 | **复杂度**: ⭐⭐

**现状**: 仅简单关键词

**功能详情**:
- `"精确匹配"` - 短语搜索
- `词1 词2` - AND
- `词1 OR 词2` - OR
- `-排除词` - NOT
- `tag:#标签` - 按标签
- `notebook:名称` - 按笔记本
- `created:2024-01-01` - 按日期
- `is:daily` `is:favorite` - 按属性

---

### 2.5 搜索替换
**优先级**: P2 | **复杂度**: ⭐⭐

**现状**: 不支持

**功能详情**:
- 快捷键 `Cmd+H`
- 当前笔记内替换
- 全局替换（需确认）
- 区分大小写、全词匹配选项
- 预览 + 逐个/全部替换

---

### 2.6 图谱视图
**优先级**: P2 | **复杂度**: ⭐⭐⭐⭐

**现状**: 无

**功能详情**:
- 全局图谱: 所有笔记和链接关系
- 局部图谱: 当前笔记的关联
- 节点大小 = 链接数量
- 节点颜色 = 笔记本
- 点击打开笔记
- 悬停预览
- 缩放、拖拽

**依赖**: `d3-force` 或 `vis-network`

## 第三阶段：数据组织

### 3.1 嵌套文件夹
**优先级**: P1 | **复杂度**: ⭐⭐

**现状**: 笔记本只有一级

**功能详情**:
- 笔记本支持多级嵌套
- 侧边栏树形展示
- 拖拽调整层级
- 折叠/展开

**数据库**:
```sql
ALTER TABLE notebooks ADD COLUMN parent_id TEXT REFERENCES notebooks(id);
```

---

### 3.2 嵌套标签
**优先级**: P1 | **复杂度**: ⭐⭐

**现状**: 标签扁平

**功能详情**:
- 语法: `#parent/child/grandchild`
- 侧边栏树形显示
- 点击父标签显示所有子标签笔记

**数据库**:
```sql
ALTER TABLE tags ADD COLUMN parent_id TEXT REFERENCES tags(id);
```

---

### 3.3 模板系统
**优先级**: P1 | **复杂度**: ⭐⭐

**现状**: 无

**功能详情**:
- 保存笔记为模板
- 新建时选择模板
- 模板变量: `{{date}}` `{{time}}` `{{title}}`
- 内置模板: 会议记录、日记、读书笔记

---

### 3.4 别名 (Aliases)
**优先级**: P2 | **复杂度**: ⭐⭐

**现状**: 不支持

**功能详情**:
- 为笔记设置多个别名
- `[[别名]]` 能正确链接
- 搜索/快速切换匹配别名

**数据库**:
```sql
CREATE TABLE note_aliases (
  note_id TEXT NOT NULL,
  alias TEXT NOT NULL,
  PRIMARY KEY (note_id, alias)
);
```

---

### 3.5 自定义属性
**优先级**: P3 | **复杂度**: ⭐⭐⭐

**现状**: 仅固定字段

**功能详情**:
- 笔记添加自定义键值对
- 类型: 文本、数字、日期、复选框
- 属性面板编辑
- 可用于搜索过滤

---

### 3.6 智能文件夹
**优先级**: P3 | **复杂度**: ⭐⭐

**现状**: 无

**功能详情**:
- 保存搜索条件为智能文件夹
- 实时显示符合条件的笔记
- 预设: 最近修改、有待办、孤立笔记

## 附录：数据库迁移汇总

```sql
-- 嵌套文件夹 (3.1)
ALTER TABLE notebooks ADD COLUMN parent_id TEXT REFERENCES notebooks(id);

-- 嵌套标签 (3.2)
ALTER TABLE tags ADD COLUMN parent_id TEXT REFERENCES tags(id);

-- 别名 (3.4)
CREATE TABLE note_aliases (
  note_id TEXT NOT NULL,
  alias TEXT NOT NULL,
  PRIMARY KEY (note_id, alias),
  FOREIGN KEY (note_id) REFERENCES notes(id) ON DELETE CASCADE
);

-- 自定义属性 (3.5)
CREATE TABLE note_properties (
  note_id TEXT NOT NULL,
  key TEXT NOT NULL,
  value TEXT,
  type TEXT DEFAULT 'text',
  PRIMARY KEY (note_id, key),
  FOREIGN KEY (note_id) REFERENCES notes(id) ON DELETE CASCADE
);

-- 版本历史 (5.3)
CREATE TABLE note_versions (
  id TEXT PRIMARY KEY,
  note_id TEXT NOT NULL,
  content TEXT NOT NULL,
  created_at TEXT NOT NULL,
  FOREIGN KEY (note_id) REFERENCES notes(id) ON DELETE CASCADE
);

-- 模板 (3.3)
CREATE TABLE templates (
  id TEXT PRIMARY KEY,
  name TEXT NOT NULL,
  content TEXT NOT NULL,
  created_at TEXT NOT NULL
);

-- 笔记加密 (5.4)
ALTER TABLE notes ADD COLUMN is_encrypted INTEGER DEFAULT 0;
ALTER TABLE notes ADD COLUMN password_hint TEXT;

-- 智能文件夹 (3.6)
CREATE TABLE smart_folders (
  id TEXT PRIMARY KEY,
  name TEXT NOT NULL,
  query TEXT NOT NULL,
  icon TEXT,
  order_index INTEGER NOT NULL,
  created_at TEXT NOT NULL
);
```
